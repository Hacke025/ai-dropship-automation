# AI Dropship Automation

This is a free, fully local AI-powered dropshipping system.

## How to Run

1. Install Python 3.10+
2. Install packages:
   pip install -r requirements.txt

## Step 1: Scrape products
python scraper/scraper.py

## Step 2: Generate AI content
python ai_content/generate.py

## Step 3: Start store
Open storefront/index.html in browser  
(or enable GitHub Pages)
